# Password Generator 🔐

A simple command-line password generator built in Python.

## Features
- Choose password type (letters/numbers/special/all)
- Custom password length
- Random password every time

## How to Run
1. Clone the repository
2. Run `python main.py`

## Usage
letters/numbers/special/all: all
Enter the length of your password: 12
Output: k3#mP9!xQ2@n
## Concepts Used
- Loops
- Conditionals
- Random module
- String manipulation
